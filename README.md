# KIN Utility System - Lab 1

## Project Overview
KIN is a campus-based utility platform designed for **Peer-to-Peer Item Sharing** and **Lost & Found** management. This project implements a secure authentication system using Spring Boot and ReactJS.

## Repository Structure
- `/backend`: Spring Boot API with MySQL and BCrypt
- `/web`: ReactJS Frontend for User Authentication
- `/docs`: Project documentation including FRS and ERD
- `TASK_CHECKLIST.md`: Progress tracking with commit hashes

## Tech Stack
- **Backend:** Java 17, Spring Boot 3.x, Spring Security, JPA/Hibernate
- **Frontend:** ReactJS, Axios, React Router
- **Database:** MySQL Workbench

## How to Run
1. **Backend:** Configure `application.properties`, then run `./mvnw spring-boot:run`.
2. **Frontend:** Navigate to `/web`, run `npm install`, then `npm start`.
